# 간단한 유전 알고리즘

python, turtle 라이브러리를 이용하여 구현

- 한마리 당 유전인자 수 : 43개
- 한 세대의 개체수 : 100마리
- 돌연변이율 : 3.5%로 시작해서 각 세대를 지날때마다 0.23%씩 감소
- 교배방식 : 이전세대 상위 10%의 유전자끼리 교배
